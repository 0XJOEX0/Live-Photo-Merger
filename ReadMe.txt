About
-------------------------------------
Live Photo Merger automatically pairs images and videos
into proper Apple-compatible Live Photos. It converts
photos to JPG and videos to MOV, then organizes them neatly
for easy import into Apple Photos.

=====================================
Features
-------------------------------------
- Auto matches photos & videos by filename
- Scans subfolders
- Converts images â†’ JPG
- Converts videos â†’ MOV
- Creates folders: merged, unmerged_images, unmerged_videos
- Auto-downloads FFmpeg
- Options: copy merged, copy unmerged, delete originals

Usage
-------------------------------------
1. Run the EXE
2. Select the three folders:
   - Image Folder
   - Video Folder
   - Output Folder
3. Choose your options
4. Click "Start Merge"
5. Import the "merged/" folder into Apple Photos
